Установка KDE
sudo add-apt-repository ppa:kubuntu-ppa/backports -y
Варианты установки
1.
sudo apt install kde-full -y
Это наиболее полный пакет, включающий все приложения KDE и основной рабочий стол KDE Plasma.
2.
sudo apt install kde-standard
Этот пакет включает в себя основной рабочий стол KDE Plasma и набор стандартных приложений, таких как веб-браузер, текстовый редактор и почтовый клиент. Это сбалансированный вариант, который обеспечивает необходимую функциональность, не потребляя так много системных ресурсов, как KDE Full. 
3.
sudo apt install kde-plasma-desktop
Этот минимальный пакет содержит только ядро KDE Plasma Desktop и несколько основных приложений. Это идеальный выбор для систем с ограниченными ресурсами или для пользователей, предпочитающих экономную установку.

Как запустить переустановку без сохранения данных
Если нет возможности зайти в БИОС
Вставить флешку с образом CloneZillaLiveCD
При начально загрузке нажать на клавиатуре комбинацию клавиш Ctrl + x
автоматически пойдёт загрузка с флеш носителя CloneZilla
выбираете образ и заливаете новую систему (с локального жесткого диска система зальётся быстрее, чем по сети)

Как запустить переустановку с сохранения данных
Если нет возможности зайти в БИОС
Вставить флешку с образом LiveCDlinmint
При начально загрузке нажать на клавиатуре комбинацию Ctrl + x
автоматически пойдёт загрузка с флеш носителя LiveCDlinmint
Выбираете образ загрузки LiveCD 64-bit

Какие программы вам нужны на запущенной системе Linux Mint Wizard LiveCD:
ПУСК -> Адмнистрирование -> Восстановление загрузчика 

Если восстановление загрузчика не помогло, тогда переходим на копирование данных с диска
ПУСК -> Системные -> Резервные копирования (перед запуском запустить рез. копирования, настройте путь куда вы будите копировать данные)

Как данные скопировали, запускайте CloneZilla
ПУСК -> Адмнистрирование -> Clonezilla


P.s. Если вам позволяет система зайти в БИОС, запускайте флешки из него.
P.s. Если ругается исчерпаны попытки входа администратора в БИОС, сбросить его через комбинацию клавиш Ctrl + r

Нажимаем F1 или F2 для входа в биос (держим до поределенного сигнала)
Во вкладке safeboot сброс предупреждений выставляем Да
Далее входим в операционную систему с помощью загрузочной флешки и комбинацией клавишь Ctrl + x
P.s. Если ошибка при входе смотрите журнал с ошибкой 000000Е  значит при входе в систему скопируйте папку с EFI в корень диска.

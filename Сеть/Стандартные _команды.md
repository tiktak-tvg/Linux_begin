##### Проверка доступности сети и ее качества.

```
ping rambler.ru
ping ya.ru
ping 8.8.8.8
ping 8.8.4.4

```

##### Определение маршрута сетевых пакетов.

```
tracepath rambler.ru
tracepath 8.8.8.8
или
apt install inetutils-traceroute 
apt install traceroute           

traceroute ya.ru
traceroute 8.8.8.8
```
